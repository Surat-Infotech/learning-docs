# Next.js Course — From Basics to Production

Welcome! 👋 This is a complete, beginner-friendly Next.js course built on the
modern **App Router** (Next.js 14/15). It takes you from "I know some JavaScript"
all the way to "I can build and ship a real, full-stack web app."

Every lesson is written in **simple, easy English**. You should be comfortable
with basic JavaScript first (variables, functions, arrays, objects, async/await).
If you are not, do the [JavaScript Course](../javascript-course/README.md) before
this one. You do **not** need to know React — Module 1 teaches the React you need.

> **Want a schedule?** Follow the
> [8-Week Weekly Roadmap (15 hours/week)](WEEKLY-ROADMAP.md). It maps every lesson
> into weeks with a project each week.

---

## How to Use This Course

1. **Go in order.** Each module builds on the one before it.
2. **Type every example.** Run a real Next.js app and edit it as you read.
3. **Keep the dev server running** (`npm run dev`) and watch your changes live.
4. **Do the exercises** at the end of each lesson before moving on.
5. **Read the terminal and browser console.** Error messages are hints.

**Suggested pace:** Spend about 40% of your time reading and 60% writing code.

## What You Will Be Able to Do at the End

- Build multi-page apps with the App Router, layouts, and dynamic routes.
- Understand and use Server Components, Client Components, and Server Actions.
- Fetch and cache data, stream slow content, and mutate data with forms.
- Style your app, optimize images and fonts, and handle SEO metadata.
- Connect to a database, manage environment variables, and add authentication.
- Handle errors, optimize performance, test, and deploy to production.

When you finish, level up with the
[Advanced Next.js Course](../nextjs-advanced-course/README.md) for senior-level
architecture, performance, and scaling topics.
