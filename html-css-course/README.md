# HTML & CSS Course — From Basics to Job-Ready

Welcome! 👋 This is a complete, beginner-friendly course on **HTML and CSS** — the
two languages every website is built from. HTML gives a page its **content and
structure**; CSS gives it **style and layout**. Together they turn plain text into
the beautiful, responsive pages you use every day.

Every lesson is written in **simple, easy English**. You do not need any prior
experience. You don't even need to know JavaScript. Just go in order, type the
examples yourself, open them in a browser, and do the exercises.

> **In a hurry, or want a schedule?** Follow the
> [2-Week Intensive Roadmap (36 hours/week)](WEEKLY-ROADMAP.md). It tells you *what*
> to study *when*; this course teaches you the *how* in detail.

---

## How to Use This Course

1. **Go in order.** Each module builds on the one before it.
2. **Type every example.** Do not copy-paste. You learn by typing and tweaking.
3. **Open your work in a browser.** Save the file, refresh the page, and watch it change.
4. **Do the exercises** at the end of each lesson before moving on.
5. **Use the browser DevTools** (press `F12`) to inspect and experiment with any page.

**Suggested pace:** Spend about 30% of your time reading and 70% building pages.

## What You Will Be Able to Do at the End

- Write clean, semantic HTML for any kind of page.
- Build forms that collect and validate user input.
- Style pages with CSS: colors, fonts, spacing, borders, and backgrounds.
- Create modern layouts with Flexbox and Grid.
- Make pages **responsive** so they look great on phones, tablets, and desktops.
- Add polish with transitions, transforms, and animations.
- Debug pages with browser DevTools and follow professional best practices.
- Turn a design mockup into a real, working web page.

When you finish, you will be ready to learn **JavaScript** and then a framework
like **React** or **Next.js**.

Happy building! 🎨
