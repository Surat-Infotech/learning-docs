# Learning Docs

A collection of free, beginner-friendly **learning courses** written in simple,
easy English. Each course takes you from "I've never done this" to confident and
job-ready, with hands-on exercises and a weekly study plan.

Every course follows the same shape:

- A **`README.md`** with the full course map.
- A **`WEEKLY-ROADMAP.md`** with a paced, week-by-week study plan.
- Numbered **modules** (`00-…`, `01-…`) containing short, focused lessons that end
  with exercises.

---

## Courses

### 🌿 [Git Course — From Basics to Advanced](git-course/README.md)

Version control from "I've never used Git" to confidently collaborating on any team,
recovering from mistakes, and keeping a clean history.

- **7 modules · 31 lessons** · [6-week roadmap](git-course/WEEKLY-ROADMAP.md) (10 hrs/week)
- **Start here if:** you're new to coding or version control.

### 🟨 [JavaScript Course — From Basics to Advanced](javascript-course/README.md)

The language of the web, from variables and functions to async, modules, and modern
ES features — the foundation for front-end and back-end development.

- **8 modules · 42 lessons** · [8-week roadmap](javascript-course/WEEKLY-ROADMAP.md) (30 hrs/week)
- **Start here if:** you want to learn programming or prepare for React/Next.js.

### ▲ [Next.js Course — From Basics to Production](nextjs-course/README.md)

Build and ship full-stack React apps with the App Router — routing, rendering, data
fetching, and deploying to production.

- **7 modules · 31 lessons** · [8-week roadmap](nextjs-course/WEEKLY-ROADMAP.md) (15 hrs/week)
- **Start here if:** you know some JavaScript/React and want to build real apps.

### ▲ [Advanced Next.js Course — For Junior and Senior Developers](nextjs-advanced-course/README.md)

Go deep: performance, architecture, advanced data patterns, and production concerns
for developers who already know the basics.

- **7 modules · 30 lessons** · [10-week roadmap](nextjs-advanced-course/WEEKLY-ROADMAP.md) (12 hrs/week)
- **Start here if:** you've finished the Next.js course and want mastery.

### 🐛 [QA Testing Course — Manual QA from Zero to Job-Ready](qa-testing-course/README.md)

Manual Quality Assurance from the ground up — testing concepts, test case design,
bug reporting, tools, and interview prep. No coding required.

- **7 modules · 31 lessons** · [4-week intensive roadmap](qa-testing-course/WEEKLY-ROADMAP.md) (~20 hrs/week)
- **Start here if:** you want a tech career that doesn't require heavy coding.

## Suggested Learning Paths

Not sure where to start? Pick the path that matches your goal.

**🧑‍💻 Become a web developer**

```text
Git  →  JavaScript  →  Next.js  →  Advanced Next.js
```

**🔍 Get into QA / software testing**

```text
QA Testing  →  Git  →  (optional) JavaScript → automation
```

**🧰 Just want the essentials every developer needs**

```text
Git  →  JavaScript
```

## How to Use These Courses

1. **Go in order.** Each module builds on the one before it.
2. **Do the exercises.** They turn reading into real skill.
3. **Practice for real** — type the commands, write the code, test the apps.
4. **Follow a roadmap** if you want a schedule; each course has one.
