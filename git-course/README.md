# Git Course — From Basics to Advanced

Welcome! 👋 This is a complete, beginner-friendly Git course. It takes you from
"I have never used version control" all the way to "I can confidently use Git on
any team, recover from mistakes, and keep a clean project history."

Every lesson is written in **simple, easy English**. You do not need any prior
experience with Git. Just go in order, type out every command yourself in a real
terminal, and do the exercises.

> **Want a schedule?** Follow the
> [6-Week Weekly Roadmap (10 hours/week)](WEEKLY-ROADMAP.md). It tells you *what*
> to practice *when*; this course teaches you the *how* in detail.

---

## How to Use This Course

1. **Go in order.** Each module builds on the one before it.
2. **Type every command.** Do not just read — run the commands in a test repo.
3. **Make a sandbox repo** (`git init practice`) and experiment freely. You cannot
   break anything important there.
4. **Do the exercises** at the end of each lesson before moving on.
5. **Read what Git tells you.** `git status` and error messages are hints, not
   punishment.

**Suggested pace:** Spend about 30% of your time reading and 70% running commands.

## What You Will Be Able to Do at the End

- Track changes to any project with confidence.
- Use branches to work on features and fixes in isolation.
- Push to and pull from GitHub, and collaborate through pull requests.
- Resolve merge conflicts calmly.
- Shape a clean, readable history with rebase, squash, and cherry-pick.
- Recover "lost" work with the reflog.
- Follow professional workflows and best practices on any team.

Happy committing! 🚀
