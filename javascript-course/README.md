# JavaScript Course — From Basics to Advanced

Welcome! 👋 This is a complete, beginner-friendly JavaScript course. It takes you
from "I have never written code" all the way to "I can build real apps and read
advanced JavaScript with confidence."

Every lesson is written in **simple, easy English**. You do not need any prior
programming knowledge to start. Just go in order, type out the examples yourself,
and do the exercises.

> **In a hurry, or want a schedule?** Follow the
> [8-Week Weekly Roadmap (30 hours/week)](WEEKLY-ROADMAP.md). It tells you *what*
> to study *when*; this course teaches you the *how* in detail.

---

## How to Use This Course

1. **Go in order.** Each module builds on the one before it.
2. **Type every example.** Do not copy-paste. You learn JavaScript by typing it.
3. **Do the exercises** at the end of each lesson before moving on.
4. **Use the browser console** (press `F12` in your browser) to try small ideas.
5. **Read error messages.** They are hints, not punishment.

**Suggested pace:** Spend about 40% of your time reading and 60% writing code.

## What You Will Be Able to Do at the End

- Read and write modern (ES6+) JavaScript with confidence.
- Build interactive web pages that respond to the user.
- Fetch data from real APIs and show it on the screen.
- Organize code into clean, reusable modules.
- Use Git, npm, and browser DevTools like a working developer.
- Understand advanced topics like closures, `this`, prototypes, and async code.

When you finish, you will be ready to learn a framework like **React**.

Happy coding! 🚀
